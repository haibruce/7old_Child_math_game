# 怪物算術大冒險

## 專案說明
這是一個兒童數學遊戲，結合了加減法練習與趣味的冒險遊戲元素。玩家需要解答算式題目來消滅怪物，並挑戰魔王。

## 功能特色

### 遊戲模式
- **加法模式**：進行加法練習
- **減法模式**：進行減法練習

### 難度設定
- **簡單模式 (1-10)**：基礎數字範圍
- **普通模式 (1-50)**：中級難度
- **困難模式 (1-100)**：高級挑戰

### 遊戲機制
- **怪物消滅**：正確回答數學題目消滅怪物並獲得分數
- **生命值系統**：玩家有3點生命值，怪物到達底部會扣除生命
- **炸彈道具**：消滅所有普通怪物，回復生命值和獲得道具獎勵
- **BOSS挑戰**：每500分出現一個魔王，需要答對10次才能打敗
- **加速功能**：加快遊戲速度，提升難度和得分倍率
- **檔位系統**：調整遊戲加速倍數（2x ~ 32x）

### 進度獎勵
- **階段獎勵**：每200分達到下一階段，遊戲難度提升
- **炸彈獎勵**：每100分獲得一顆炸彈
- **BOSS獎勵**：打敗魔王恢復3點生命值並獲得1顆炸彈

## 使用說明

### 開啟遊戲
直接用瀏覽器開啟 `Child_plus_game.html`

### 遊戲流程
1. 選擇數學模式（加法或減法）
2. 選擇難度等級
3. 點擊正確答案消滅怪物
4. 積累分數並挑戰魔王
5. 當生命值歸零時遊戲結束

### 控制方式
- **點擊答案按鈕**：選擇正確答案
- **炸彈按鈕**：消滅所有普通怪物（需要炸彈數量 > 0）
- **加速按鈕**：開啟/關閉加速模式
- **檔位排檔**：調整加速倍數
- **返回按鈕**：返回主選單

## 技術架構

### 技術棧
- **HTML5**：頁面結構與Canvas繪圖
- **Tailwind CSS**：樣式美化
- **原生JavaScript**：遊戲邏輯與動畫

### 核心組件
- **Canvas 遊戲渲染引擎**：繪製怪物、粒子效果、背景動畫
- **事件系統**：按鈕點擊、答題判定
- **數據管理**：分數、生命值、炸彈、遊戲狀態
